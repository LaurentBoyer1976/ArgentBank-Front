# Argent Bank Frontend

Une application moderne et prête pour la production, construite avec React et React Router.

## Fonctionnalités

- 🚀 Rendu côté serveur (Server-side rendering)
- ⚡️ Hot Module Replacement (HMR) pour un développement rapide
- 📦 Optimisation et bundling des assets
- 🔄 Chargement et mutations de données
- 🎨 Styling avec Sass
- 📖 [Documentation React Router](https://reactrouter.com/)
